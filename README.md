# Sorte RPA
![Captura de Tela 2024-07-04 às 1 15 29 AM](https://github.com/NatanRubenich/Sorte-RPA/assets/57855715/facfed9a-5ab6-4b92-9e89-970c804d9e15)

### Contexto
App produzido para automatizar a votação de mérito empresarial. Por ter diversas empresas de diversos ramos, a votação era exaustiva. 

### Objetivo
Fazer a votação de forma automática e randomicamente. Quando identificar a determinada empresa que queria votar, passa por diversas checagem e vota automaticamente.
É necessario fornecer usuário e senha no Swegger da API.

### Tecnologias
* Selenium
* FastAPI
